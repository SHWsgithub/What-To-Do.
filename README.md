# 개인용 투두 앱 What To Do.
- Personal, Business 일정을 추가, 불러오기, 수정, 삭제 가능
- 광역 상태 저장소에 Personal, Business 부분으로 구분하여 저장
- 각 일정의 완료, 미완료 상태 저장
- LocalStorage에 보관하여 웹에서 개인용으로 사용 가능
- react
- redux
- react-redux
- styled-Components
